# CTMCEnsemble.average — Function.

average(preds, weights=nothing; multiplicity=true)

Compute the average. multiplicity for arithmetic mean based on mulitpiliciity.

Example

julia> average([([0.5, 0.5], [1, 2]), ([0.5, 0.5], [2, 3])])
